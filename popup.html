Aviso: Esta extensão atualmente suporta apenas o Português (Brasil). Ao clicar no botão, você poderá coletar todos os cupons disponíveis na tela atual do Mercado Livre.

Para obter mais cupons, será necessário navegar para outras páginas.

Além disso, se o site estiver usando medidas de segurança para prevenir esse tipo de comportamento, o script pode não funcionar. Portanto, use este script com cautela.

Para evitar erros, esteja logado no mercado livre.
